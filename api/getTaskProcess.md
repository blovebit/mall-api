# 查看订单进度详情: getTaskProcess

- 查询**设计师与订单的关联表**  
  以设计师为查询条件，查看订单的步骤详情  
  返回 施工图交付凭证、尾款支付凭证、终稿、定金支付凭证、初稿 5个步骤的详细信息  
  未进行到的步骤，请勿返回该字段

## 参数

    {
        "task_id": 1, // 订单ID
        "designer_id": 1 // 设计师ID
    }
